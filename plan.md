# ПЛАН МАСШТАБИРОВАНИЯ ТУРАГЕНТСТВА
**Владелец:** Руководитель турагентства (Уфа)  
**Дата создания:** 09.04.2026  
**Главная цель:** Быстрые деньги + автоматизация + масштабирование

---

## КОНТЕКСТ БИЗНЕСА

### Текущая ситуация
- Турагентство в Уфе
- 3 менеджера в команде
- ВК-сообщество: **5700 подписчиков** (vk.com/family_toor)
- MAX-канал: **Пегас Туристик Уфа** (~300 подписчиков)
- База клиентов: **500–2000 человек**
- CRM: **"Мои документы туризм"** (есть API)
- Турвизор: есть логин агентства

### Ключевые боли
- Нет времени на публикацию горящих туров — упускаем продажи
- ВК-реклама: 10 000₽ → 20 лидов → 0 продаж (холодная аудитория)
- Много времени на ручной контент: поиск фото, написание текстов, слежка за конкурентами
- Большие расходы на бухгалтера
- 3 менеджера требуют постоянного контроля (CRM, общение, отчёты)
- Спад из-за ситуации на Ближнем Востоке, не хватает на зарплаты
- Всё держится на руководителе — нет системы

### Финансовые параметры
- Рекламный бюджет сейчас: **5–10к₽/месяц**
- Хостинг: готова платить **0–500₽/месяц**

## УРОВЕНЬ 1 — БЫСТРЫЕ ДЕНЬГИ

### 🔥 ПРОЕКТ #1: MAX-бот горящих туров (ГЛАВНЫЙ, СРОЧНО)
**Спецификация:** `thoughts/shared/specs/2026-04-09-hot-tours-bot.md`  
**Код:** `hot-tours-bot/` — написан полностью, архитектура пересмотрена  
**Статус:** 🔄 Смена источника данных → переходим на Google Sheets + RSS

**Что делает:**
- Парсит горящие туры с Турвизора по фильтрам (цена, скидка, направление, даты, города вылета)
- Публикует 2-3 раза в день в ВК + MAX + Telegram с фото, описанием, преимуществами
- Бот собирает заявки: имя, телефон, тур, даты, состав, бюджет
- Хранит базу клиентов для догрева и рассылок
- Уведомляет руководителя → менеджер звонит

**Технический стек:** Python, Tourvisor API, VK API, Telegram Bot API, Claude API (Haiku), Google Sheets, Railway (хостинг)  
**Стоимость:** ~100–500₽/месяц  
**Цель:** 1–2 продажи в первый месяц

**Решение по источнику данных (консилиум агентов 10.04.2026):**

Tourvisor API поиска — 5000₽/мес (отказались). Парсинг Pegas/Sletat — блокируют по IP.

**Принятая стратегия — три этапа:**

| Этап | Что | Срок |
|------|-----|------|
| 1 (сейчас) | Google Sheets → бот публикует | День 1 |
| 2 | RSS-агрегатор (tourdom.ru, turpravda.com) → черновики | День 3-5 |
| 3 | Telethon мониторинг каналов Пегас/TEZ/Coral | Неделя 2 |

**Отклонено:** скрытые XHR API (нестабильно), мобильный API через прокси (юридический риск), 7 B2B систем сразу (регистрация 3-14 дней каждая)

**Что нужно для запуска (этап 1):**
- [ ] Переписать `main.py` под источник Google Sheets
- [ ] Добавить лист "Туры к публикации" в Sheets
- [ ] Создать Telegram-бота через @BotFather → получить токен
- [ ] Добавить бота как администратора в канал
- [ ] Заполнить `.env` и запустить первый пост

**Формат поста:**
```
🔥 ГОРЯЩИЙ ТУР! Турция, Анталья
✈️ Вылет: 15 апреля из Уфы (7 ночей)
🏨 Отель: Rixos Premium Belek 5⭐
🍽 Питание: Ultra All Inclusive
🌊 До моря: 50 метров
💰 Цена: 42 000 ₽/чел  ~~60 000₽~~  (-30%)
✅ Первая линия | ✅ Аквапарк | ✅ Анимация для детей
📩 [кнопка → бот для заявки]
⚡ Осталось 2 места!
```
