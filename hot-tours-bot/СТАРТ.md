# СТАРТ — Запуск Family Tour Bot

Это пошаговая инструкция для запуска бота с нуля.
Код уже написан. Твоя задача — только заполнить токены и нажать запуск.

---

## Что нужно сделать ОДИН РАЗ (подготовка)

### 1. Установи Python

1. Зайди на [python.org/downloads](https://python.org/downloads)
2. Скачай Python **3.11** или новее
3. При установке **обязательно** поставь галочку **"Add Python to PATH"**

Проверь в терминале VS Code (`Ctrl + тильда`):
```
python --version
```
Должно показать: `Python 3.11.x`

---

### 2. Открой папку проекта в терминале

```bash
cd "c:/Users/Пользователь/Documents/Проекты/moi_idei/hot-tours-bot"
```

---

### 3. Создай виртуальное окружение

> Это изолированная папка с библиотеками для этого проекта. Делается один раз.

```bash
python -m venv venv
venv\Scripts\activate
```

После активации в начале строки появится `(venv)` — это нормально.

---

### 4. Установи зависимости

```bash
pip install -r requirements.txt
```

Займёт 1–2 минуты. Один раз.

---

### 5. Создай файл настроек

```bash
copy .env.example .env
```

Открой `.env` в VS Code. Заполни токены — см. таблицу ниже.

---

## Таблица токенов — где взять каждый

| Переменная | Где получить | Статус |
|------------|-------------|--------|
| `TOURVISOR_LOGIN` | Email на tourvisor.ru (справочники работают) | ✅ Заполнено |
| `TOURVISOR_PASSWORD` | Пароль на tourvisor.ru | ✅ Заполнено |
| `TELEGRAM_BOT_TOKEN` | @BotFather → `/newbot` | ✅ Заполнено |
| `TELEGRAM_ADMIN_ID` | Написать @userinfobot | ✅ Заполнено (418012639) |
| `TELEGRAM_CHANNEL_ID` | @username канала или `-100XXXXXXXXX` | ❌ Нужно заполнить |
| `VK_TOKEN` | [vk.com/dev](https://vk.com/dev) → Мои приложения → ключ доступа группы | ❌ Нужно заполнить |
| `VK_GROUP_ID` | Числовой ID группы family_toor (без минуса) | ❌ Нужно заполнить |
| `ANTHROPIC_API_KEY` | [console.anthropic.com](https://console.anthropic.com) → API Keys | ❌ Нужно заполнить |
| `GOOGLE_CREDENTIALS_FILE` | JSON-ключ сервисного аккаунта Google (см. ниже) | ❌ Нужно настроить |
| `GOOGLE_SHEET_ID` | ID из URL таблицы Google Sheets | ❌ Нужно настроить |

> **Примечание по источнику туров:** Tourvisor API поиска платный (5000₽/мес) — не используем.
> Вместо него: менеджер вносит тур в Google Sheets за 2 минуты → бот публикует автоматически.

## Как создать Telegram-бота (5 минут)

1. Открой Telegram → найди **@BotFather**
2. Напиши `/newbot`
3. Придумай название: `Family Tour Уфа`
4. Придумай username (латиницей, оканчивается на bot): `familytour_ufa_bot`
5. BotFather пришлёт токен вида `7123456789:AAF...`
6. Скопируй в `.env` → `TELEGRAM_BOT_TOKEN=...`

---

## Как настроить Google Sheets (10 минут)

1. Зайди на [console.cloud.google.com](https://console.cloud.google.com)
2. Создай новый проект (любое название)
3. Включи два API: **Google Sheets API** и **Google Drive API**
   - Поиск по "Google Sheets API" → Включить
4. Создай сервисный аккаунт:
   - IAM и администрирование → Сервисные аккаунты → Создать
   - Скачай JSON-ключ → сохрани как `google_credentials.json` в папку проекта
5. Создай новую Google Sheets таблицу
6. Поделись таблицей с email сервисного аккаунта (из JSON-файла) как **Редактор**
7. Скопируй ID таблицы из URL и вставь в `.env` → `GOOGLE_SHEET_ID=...`

## Запуск по шагам

### Шаг 1 — Настрой Google Sheets

Смотри инструкцию ниже в разделе "Как настроить Google Sheets".

После настройки в таблице появится лист **"Туры к публикации"** с колонками:
`Статус | Страна | Курорт | Отель | Звёзды | Питание | Дата вылета | Ночей | Цена/чел | Фото URL`

---

### Шаг 2 — Добавь бота в Telegram-канал

1. Открой настройки канала → Администраторы
2. Добавь бота `@familytour_ufa_bot` (или как назвала)
3. Дай права: публикация сообщений, редактирование
4. Узнай ID канала — напиши в канал любое сообщение, потом перешли его боту @userinfobot — он покажет chat_id
5. Вставь ID канала в `.env` → `TELEGRAM_CHANNEL_ID=...`

---

### Шаг 3 — Первый тур

1. Открой Google Sheets → лист "Туры к публикации"
2. Добавь строку с данными любого тура
3. В колонке "Статус" поставь `НОВЫЙ`

---

### Шаг 4 — Запуск

```bash
python main.py
```

Бот проверяет таблицу каждые 5 минут. Находит `НОВЫЙ` → публикует → ставит `ОПУБЛИКОВАН`.

Чтобы остановить: `Ctrl + C`

---

## Режимы работы

| Режим | Как | Что происходит |
|-------|-----|----------------|
| **Одобрение** (неделя 1) | `APPROVAL_MODE = True` в `main.py` | Бот присылает превью → ты нажимаешь ✅ или ❌ |
| **Автопилот** (со 2й недели) | `APPROVAL_MODE = False` | Бот публикует сам, присылает уведомление |
